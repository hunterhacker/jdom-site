Our Mission? To build a complete, Java-based solution for accessing, manipulating, and outputting XML data from Java code. Read more about our project goals.
Project Status? We've made major strides, and now JDOMTM Beta 7 is available. Check it out and talk about it on the mailing lists today.
The Latest?

JDOM Beta 7 was released! See the news page for a summary of the improvements over Beta 6. We highly recommend upgrading.

JDOM was accepted as JSR-102! A JSR is how formal Java specifications are defined, and having JDOM as a JSR will facilitate JDOM's corporate adoption and open the door for JDOM to be incorporated into the core Java Platform. For more information, see the News page.

Jason gave a three-hour tutorial on JDOM at the O'Reilly Conference on Enterprise Java. Slides are available.

Jason gave a manager-level introduction to JDOM talk at the Sun HQ Briefing in Menlo Park. Slides are available.

Greatly expanded the FAQ. It's about twice as long as before. Also updated the Quotes section.
