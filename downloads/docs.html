JDOMTM Documentation is provided for users wanting to learn more about JDOM before downloading code, and as a reference companion to using the code. The JDOM Documentation includes the JDOM Specification and complete JavaDoc for the JDOM project, as well as information related to JDOM such as presentation slides and handouts.

For those of you looking for the JDOM binaries you can find them here.
For those of you looking for the JDOM source code you can find it here.

JDOM JavaDoc and UML
The project JavaDoc is generated on each milestone and public release. If you want to get a taste of the latest fully-supported JDOM features, this is the place to be. For up-to-the-minute JavaDoc on new features, generate the documention from the JDOM source.

JavaWorld article: "Easy Java/XML Integration with JDOM"
A two-part introduction to JDOM published by the prestigious JavaWorld magazine. The first article explains how to use JDOM to read XML from a pre-existing source. The second article explains how to create and manipulate XML using JDOM.
