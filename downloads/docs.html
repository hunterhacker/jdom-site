JDOMTM Documentation is provided for users wanting to learn more about JDOM before downloading code, and as a reference companion to using the code. The JDOM Documentation includes the JDOM Specification and complete JavaDoc for the JDOM project, as well as information related to JDOM such as presentation slides and handouts.

For those of you looking for the JDOM binaries you can find them here.
For those of you looking for the JDOM source code you can find it here.

JDOM in Oracle Magazine
This series of articles published by Oracle Magazine cover JDOM's Beta 8 release and are the most thorough and up-to-date articles.

  •   "JDOM and XML Parsing, Part 1"
  •   "JDOM and XML Parsing, Part 2"
  •   "JDOM in the Real World, Part 3"

JDOM Makes XML Easy
This talk was given by Jason Hunter at JavaOne 2002 to around a thousand people. It covers Beta 8. The slides are available on Jason's site Servlets.com.

  •   "JDOM Makes XML Easy" slides from JavaOne 2002

Processing XML with Java
Elliotte Rusty Harold's upcoming book Processing XML with Java (Addison-Wesley, 2002) contains extensive coverage of JDOM. The complete text of the book is available online at Cafe con Leche.

  •   Chapter 14: JDOM
  •   Chapter 15: The JDOM Model

JDOM JavaDoc and UML
The project JavaDoc is generated on each milestone and public release. If you want to get a taste of the latest fully-supported JDOM features, this is the place to be. For up-to-the-minute JavaDoc on new features, generate the documention from the JDOM source.

  •   UML for JDOM Beta 6 (very old).

IBM developerWorks
The developerWorks site has run a series of articles on JDOM.
Note that the code for XSLT transforms can be done more easily than shown in these articles by taking advantage of the new JDOMSource and JDOMResult classes from jdom-contrib. See the FAQ.

  • "Simplify XML programming with JDOM" by Wes Biggs and Harry Evans (May 2001)
  • "Converting from DOM" by Brett McLaughlin (April 2001)
  • "Converting from SAX" by Brett McLaughlin (April 2001)
  • "Using JDOM and XSLT" by Brett McLaughlin (March 2001)

The following is a short 45-minute talk on JDOM given by Jason at the O'Reilly Open Source Convention in San Diego on July 25, 2001. It's based on beta7 (now old) and talks about the JSR-102 effort. The talk below from March is more substantial but covers the older beta6.

JavaWorld article: "Easy Java/XML Integration with JDOM"
A two-part introduction to JDOM published by the prestigious JavaWorld magazine. The first article explains how to use JDOM to read XML from a pre-existing source. The second article explains how to create and manipulate XML using JDOM. These articles are quite old.

  •    Read Part Two of the JavaWorld article (manipulating)
